23년 3월 내 쿠폰 발송 예정

  • 응모하기 버튼 클릭 시 KT해외주식정보 유료 가입하기 페이지로 이동합니다.

  • 이벤트 기간 내 유료 가입 완료 시 자동으로 이벤트에 참여됩니다.

  • 해당 이벤트는 최초 KT해외주식정보 유료가입 회원 대상으로 진행되는 이벤트입니다.

  • KT해외주식정보 유료 가입 시 당일 해지자 제외 스타벅스 아메리카노 Tall 쿠폰 전원 증정, 유료 가입 한달 이상 유지 고객 신세계 상품권 2만원 권 전원 증정, 3개월 이상 유지 고객 대상으로 추첨을 통해 각 1명 씩 아이폰 14, 에어팟 프로 2를 증정합니다.

  • 5만원 이상 경품 수령 시 22% 제세공과금은 당첨자 본인 부담입니다.

  • 실물 경품 당첨자에게는 이벤트 참여 시 제공된 정보로 개별 연락 예정입니다.

  • 본 이벤트는 제공업체 사정에 의해 별도의 고지 없이 중단 또는 변경될 수 있습니다.

  • 본 서비스는 청약철회가 불가능한 상품입니다.

  • 해외주식정보(유료)는 매월 5,500원(부가세 포함)이 부과됩니다.
